What is css

  1. CSS stands for Cascading Style Sheets.

  2. CSS describe how HTML elements are to be displayed on screen, paper, or in other media.

  3. Css is not necessary CSS saves a lot of work. It can control the layout of multiple web pages all at once

  4. External stylesheet are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elementsl
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate , and easier for you to try it yourself.

When computing CSS class vs ID, the diffrence is that CSS class applies a style to multiple elements ID, on the other hande applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by javascript.

In CSS, selectors are used to target a specific element or range of element on a web page. Once an element has been targeted, a style or set of styles can applied to the element.

There is wide range of selectors available. Two of the most commonly used are class and ID. BBoth are used to target element to which should be applied.
